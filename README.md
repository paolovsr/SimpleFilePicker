# SimpleFilePicker
Simple library used to get a file or folder Uri


Usage:

Add repository:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
  Add dependency:
  
  	dependencies {
	        implementation 'com.github.paolovsr:SimpleFilePicker:Tag'
	}
